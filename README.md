# DocumentsStealer

A program that scan directory recursively for documents extension from a specified directory, default directory is C:\Users\%USERMANE%\ can be changed to fits your usage. 
### Default extension :
- *.doc, *.docx, *.docm
- *.ppt, *.pptx, *.pptm *.ppsx, *.ppsm
- *.sldx
- *.pdf
- *.odt
